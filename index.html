Things I Can Do

HTML and CSS (with a little JavaScript) Python C Java Ruby on Rails JavaScript, Assembly, Markdown

 

 

I have worked on various personal projects with HTML and CSS (including this webpage), but alas many of them were local files that I fooled around with and never pushed to GitHub. RIP my hard work.

My history with HTML and CSS goes back to the start of my Second Year at UVic (Fall 2014) when I joined the UVic Web Design and Development Club. A year and a half later, I sit as a Co-President of the club as well as continue to make web pages and other personal projects.

Years of Knowledge: 1.5

 

 

After learning Python in Software Engineering 265, Python became my go-to language when it came to prototyping my ideas. Due to its ease-of-use, I enjoy picking it up and being able to write short programs whenever they are needed.

During the Weekend to Code II hackathon that my club hosted last year, I worked on a barebones Sentiment Analysis program with a friend that took us 8 hours to code. The source code is available on my GitHub.

Years of Knowledge: 1.5

 

 

Originally learned in Software Engineering 265, C is collectively my most-used coding language. I personally enjoy C because, due to it being a lower-level language, I am able to ensure that my algorithms are doing exactly as they should with key memory spaces.

Although I have no personal projects written in C, I have used it in various courses such as Software Engineering 265, Computer Science 230 (Embedded Systems), Computer Science 360 (Operating Systems), and am currently using it in Computer Science 361 (Networks).

Years of Knowledge: 1.5

 

 

The first programming language I ever learned, Java provided me with all of my fundamentals for programming. Due to this, I have a strong foundation for learning and (hopefully) mastering other languages.

This really shows how far I have come since I started at UVic. When I came here a mere two and a half years ago, I had no experience with code whatsoever, and now I sit as the Co-President of the UVic Web Design and Development Club, as well as know around 8 programming languages.

Besides random assignments from courses, my favourite part of working with Java was creating Servlets for my team's Oracle database in Computer Science 370 (Databases).

Years of Knowledge: 2.5
